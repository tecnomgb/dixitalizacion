Introdución a Python

Actividade 1

Empregando o IDE de Python Thonny (aplicacións→programación→Thonny), a partir do ficheiro modelo .py adxunto máis abaixo, crea un programa que ao executarse faga aparecer en pantalla a mensaxe seguinte:

  • Unha primeira liña de texto correspondente a unha mensaxe de saúdo informando do teu nome e apelidos
  • Unha segunda liña que informe do centro, curso e grupo no que estás actualmente
  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques que este é o teu primeiro programa en Python
