Introdución a Python

Actividade 1

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ) e a partir do ficheiro modelo .py adxunto máis abaixo, crea un programa que ao executarse faga aparecer en pantalla a mensaxe seguinte:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Unha primeira liña de texto correspondente a unha mensaxe de saúdo
  • Unha segunda liña de texto informando do teu nome e apelidos.
  • Unha terceira liña que informe do centro, curso e grupo no que estás actualmente.

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
