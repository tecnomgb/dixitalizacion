Actividade 09

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ), abre o ficheiro modelo adxunto máis abaixo co nome actividade9.py e gárdao como actividade9_OTeuNome.py.

Sabendo que a letra que corresponde a un número de DNI vén dado pola posición numérica nunha determinada lista que xera o resto de dividir o número de DNI entre 23 (é dicir, o módulo 23 do número de DNI), crea a continuación un programa que cumpra os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa deben aparecer dúas mensaxes na que o programa pregunte polo número de DNI (sen letra) do usuario. O programa deberá almacenar ese número nunha variable que chamaremos num_dni.
  • A continuación o programa calculará para ese número o seu módulo 23 e asignaráo á variable resto.
  • A partir da variable resto o programa deberá achar que letra corresponde na lista letras_posibles_dni coa posición dada pola variable resto. Este resultado almacenarémolo na variable letra_dni.
  • Finalmente deberá aparecer en pantalla unha mensaxe na que informe de que letra corresponde co número introducido (na mensaxe ten que figurar tanto o número de DNI como a letra asignada).

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
