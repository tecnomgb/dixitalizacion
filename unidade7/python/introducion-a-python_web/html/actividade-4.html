Introdución a Python

Actividade 4

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ), abre o ficheiro modelo adxunto máis abaixo co nome actividade4.py. A continuación crea nel un programa que cumpra os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa debe aparecer unha primeira mensaxe na que o programa informe que é quen de realizar a suma de dous números.
  • A continuación debe aparecer en pantalla unha mensaxe preguntando ao usuario polo valor do primeiro número, que chamaremos a.
  • Despois, nunha nova mensaxe, preguntará polo valor do segundo número, que chamaremos b.
  • O programa deberá almacenar o resultado da suma dos dous números nunha variable que se chamará como ti desexes.
  • Finalmente deberá amosar unha última mensaxe indicando o resultado da suma dos dous número proporcionados polo usuario.

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
