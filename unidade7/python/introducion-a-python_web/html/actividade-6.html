Introdución a Python

Actividade 6

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ), a partir do ficheiro que entregaches na actividade anterior (actividade5-nome.py) e que gardarás co nome actividade6-nome.py, crea un programa que cumpra os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa debe aparecer unha primeira mensaxe na que o programa informe que é quen de saber se un número é par ou impar.
  • A continuación debe aparecer en pantalla unha mensaxe pedindo ao usuario que introduza un número calquera.
  • O programa deberá almacenar o valor dese número nunha variable que se chamará como ti desexes.
  • A continuación o programa debe calcular se o número é par ou impar a partir da operación matemática "módulo de 2", que é capaz de identificar se un número é par ou impar, xa que:
    • Se o resultado do "módulo de 2" dun número é 0 → ese número é par. Exemplo:
      • 6%2 lese "6 módulo de 2"
      • Dá por resultado 0
      • Polo que 6 é par
    • Se o resultado do "módulo de 2" dun número é 1 → ese número é impar. Exemplo:
      • 7%2 lese "7 módulo de 2"
      • Dá por resultado 1
      • Polo que 7 é impar
  • Almacena o resultado de calcular o módulo do número introducido nunha variable que chamarás modulo.
  • Finalmente, emprega o condicional if else para que o programa amose unha última mensaxe indicando que dito número introducido é par ou impar (segundo corresponda).

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
