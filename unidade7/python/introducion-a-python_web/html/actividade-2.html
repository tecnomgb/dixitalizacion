Introdución a Python

Actividade 2

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ) e a partir do ficheiro modelo .py adxunto máis abaixo, crea un programa que cumpra os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa debe aparecer unha primeira mensaxe na que se presente o programa saudando.
  • A continuación debe aparecer en pantalla unha mensaxe preguntando o nome ao usuario.
  • O programa deberá almacenar o nome tecleado polo usuario nunha variable que se chamará nome_usuario.
  • Finalmente deberá amosar unha última mensaxe de despedida dando as grazas pola información. No texto da mensaxe de despedida deberá aparecer o nome tecleado polo usuario, é dicir, o valor de texto contido na variable nome_usuario.

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
