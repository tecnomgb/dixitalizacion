Actividade 05

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ), abre o ficheiro modelo adxunto máis abaixo co nome actividade5.py e gárdao co nome actividade5_OTeuNome.py. A continuación crea nel un programa que cumpra os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa debe aparecer unha primeira mensaxe na que o programa informe que é quen de saber se un número é positivo ou negativo.
  • A continuación debe aparecer en pantalla unha mensaxe pedindo ao usuario que introduza un número calquera entre -100 e 100.
  • O programa deberá almacenar o valor dese número nunha variable que se chamará como ti desexes.
  • Finalmente deberá amosar unha última mensaxe indicando que dito número é positivo ou negativo (segundo corresponda).

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
