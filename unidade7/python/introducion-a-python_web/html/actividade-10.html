Actividade 10

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ), abre o ficheiro modelo adxunto máis abaixo co nome actividade10.py e gárdao como actividade10_OTeuNome.py. A continuación edita o programa para que cumpla os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa debe aparecer unha mensaxe pedindo ao usuario un número entre 0 e 100.
  • A continuación o programa calculará todos os números pares entre 0 e o número introducido polo usuario e amosaraos en pantalla.

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
