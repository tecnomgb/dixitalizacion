Actividade 11

Empregando o IDE de Python chamado Thonny ( Aplicacións → Programación → Thonny ), abre o ficheiro modelo adxunto máis abaixo co nome actividade11.py e gárdao como actividade11_OTeuNome.py. A continuación edita o programa para que cumpla os seguintes requisitos:

  • Ao comezo do ficheiro fonte deberá haber un comentario no que indiques o teu nome e apelidos, curso e grupo e número de actividade.
  • Ao executar o programa debe aparecer unha mensaxe pedindo ao usuario un número calquera.
  • A continuación o programa informará en pantalla se ese número é par ou impar.
  • Finalmente o programa preguntará se se quere probar con outro número ou rematar o programa. No caso de querer rematar o programa deberá amosar en pantalla unha mensaxe de despedida e a continuación rematar a execución.

Ao rematar de editar o código e comprobar o correcto funcionamento do teu programa, súbeo á correspondente tarefa da aula virtual.
